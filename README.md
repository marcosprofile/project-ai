# Figma: Construindo um produto com IA e MCP

Neste curso, trabalhamos em nosso projeto no Figma, começando pela parte mais simples e rápida, que é criar a interface, o protótipo, as variáveis e os componentes. Em seguida, avançamos para a parte mais complexa e exigente, que é a parte tecnológica, envolvendo o MCP e o uso de Inteligência Artificial (IA).

<hr>

- [x] Converter protótipos do Figma em páginas HTML/CSS aplicando o MCP e agentes de IA.
- [x] Estruturar interfaces responsivas usando auto layout, grids e variáveis de estilo.
- [x] Exportar e integrar ícones SVG, bibliotecas de ícones e fontes para aprimorar o visual do projeto.
- [x] Definir e aplicar tokens e variáveis globais de CSS para garantir consistência visual.
- [x] Aproveitar recursos do VS Code e assistentes de IA para automatizar a criação de componentes e estilos.
- [x] Integrar o projeto ao GitHub para versionamento e publicação colaborativa.
